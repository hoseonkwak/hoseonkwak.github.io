gitBash 설치 및 간단한 CLI 명령어

작성일 2022-05-01 In git

깃 설치

깃 공식홈페이지에서는 없는 도구를 사용해야하기 때문에

https://gitforwindows.org/ 에서

다운로드를 해줍니다.

CLI를 사용할거기 때문에 Gitbash로 작업합니다.

CLI란 Command-Line Interface의 약자로 명령 줄 인터페이스 입니다.
가상 터미널 또는 터미널을 통해 사용자와 컴퓨터가 상요 작용하는 방식을 말합니다.

- git bash 실습

Gitbash를 실행해 git –version을 입력하고 버전이 나오면 설치 완성!

맨 뒤에 ~ 은 최상위를 의미합니다.
($, >, %)는 명령을 받을 준비가 됐다는 의미입니다.

- CLI에서 자주 사용하는 명령어

1. ls(list segment)

현재 디렉토리의 파일 목록을 확인하는 명령어 입니다.

2. ls -a

현재 디렉토리의 숨긴파일(.으로 시작)도 확인하는 명령어 입니다.

3. ls -l

상세정보를 확인하는 명령어 입니다.

4. ls -al

숨긴파일을 포함한 상세정보를 확인하는 명령어 입니다.

5. cd(change directory)

디렉토리를 이동하는 명령어 입니다.

6. ‘..’

상위 디렉토리로 올라간다.

7. ‘.’

‘.’은 현재 디렉토리를 의미한다.

8. mkdir

폴더를 만드는 명령어 입니다.

9. touch

파일 만드는 명령어 입니다.

  • 여러개 동시에 만들때는 &&를 중간에 입력합니다.

10. mv(move)

파일 이동 또는 이름을 변경하는 명령어 입니다.
mv OO를 OO로

다른위치에 있는 파일을 현재 위치로

와일드카드(*)를 이용한 방법

이름 변경

11. cp(copy)

복사하는 명령어 입니다.

12. rm(remove)

삭제하는 명령어 입니다.

rm -r이 안될때는 rm -rf 를 입력합니다.
